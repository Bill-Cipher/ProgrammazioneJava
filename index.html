Welcome to GitHub Pages.

Questa pagina presenta il materiale didattico per il corso di Programmazione Java di Fausto Spoto presso il Dipartimento di Informatica dell'Università di Verona. Per copiare l'intero materiale didattico sulla vostra macchina locale, istallate git ed eseguite il comando:

$ git clone git@github.com:spoto/ProgrammazioneJava.git
